# 🚚 Fulfillment Guide

Next Commerce offers a deep feature set related to order fulfillment.  This guide, and the pages that follow, offer an overview of fulfillment features.

## Core Concepts

[**Products**](../../build-a-store/catalogue/) in your store may be sold and fulfilled digitally, or correspond to physical [**Inventory**](../../build-a-store/catalogue/inventory.md) items (SKUs) located at one or more [**Fulfillment Locations**](../../start-here/get-started/fulfillment-settings.md#fulfillment-locations).

Product pricing is set on a per-currency level, and one product can be sold in multiple locales via multiple fulfillment locations.

Orders containing multiple line items may have items fulfilled from different locations (split fulfillment).&#x20;

## Setting up Fulfillment

Head to the [**Fulfillment Settings**](../../start-here/get-started/fulfillment-settings.md) guide to set up shipping methods, fulfillment locations, and returns handling for your store.
