# Stripe APMs

Next Commerce supports several regional alternative payment methods both in the storefront checkout and on custom checkouts on the Admin API:

* Klarna Payments
* iDEAL
* Bancontact
* Sofort Banking
* Giropay
* SEPA Debit

These methods, along with [**Apple Pay**](apple-pay.md)**,** are available to enable as sub-methods under **Stripe** gateways, from the **Alternative Payment Methods** card.

Note - these methods may be available in specific regions only.  Consult Stripe's documentation to determine which methods are available in each market.

{% hint style="info" %}
Visit the [**Alternative Payment Methods**](https://developers.29next.com/docs/api/admin/guides/stripe-apms/) developer guide for more information on implementation on custom checkouts via the Admin API.
{% endhint %}
