# Subscription Statuses

## **Subscription Statuses**

Subscription Statuses represent various points in a subscriber’s lifecycle.

| Status       | Description                                                                                |
| ------------ | ------------------------------------------------------------------------------------------ |
| **Active**   | Subscription is Active and is scheduled to renew at the set Next Renewal Date.             |
| **Retrying** | Subscription could not be renewed and is currently in an automated recovery process state. |
| **Past Due** | Subscription could not renew due to a failed recurring payment attempt.                    |
| **Canceled** | Subscription is canceled and will not renew.                                               |
