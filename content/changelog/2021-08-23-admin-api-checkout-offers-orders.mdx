## New Features

- Introducing support for a PayPal "One Click" checkout flow on the Admin Orders API allowing customers to checkout directly on PayPal.

## Improvements

- We've now updated the [Midigator](https://docs.nextcommerce.com/docs/apps/midigator) Extension to allow store admins to add their Midigator MID to map with store gateways so that the correct MID is synced to Midigator with transactions.
- Improved the Date filters in [Admin API](https://developers.nextcommerce.com/docs/admin-api) Swagger documentation to be more clear for use and expected date formats.

## Bug Fixes

- We've now corrected some Omnisend (legacy email marketing app) syncing issues caused by rapid succession of tasks with dependencies being processed concurrently.
- We've now corrected an issue that prevented offers and vouchers from being passed and applied together on the Admin Orders API.
