## New Features

- Introducing Omnisend (legacy email marketing app) Push Notifications support.
- Shipping Links for all “Order Shipped” emails contain a link to the respective carrier tracking page.
- Language Translations are now available for content pages following most other content fields.

## Improvements

- Softwrap for Content Code Editors so there is no horizontal scrolling inside the editor.
- Updated main CTA button in sidecart to from “Continue” to “Proceed to Checkout” to be more easily understood.
- Dashboard cart line items has been improved more easy to understand the amounts that are calculated for the order total.
- Improved Address Layout in checkout flow.

## Bug Fixes

- Fixed an issue with “cheapest item for free” incentive in Offers app.
- Fixed an issue with coupon not applied when redirect to checkout.
- Fixed an issue with dashboard Customer search filter showing duplicate results for Dashboard user searches.
