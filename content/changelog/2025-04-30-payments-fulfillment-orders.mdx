## **Improvements**

- **Simplified Test Order Creation:** Test orders can now be created even before any gateways are configured in the store’s default gateway group, streamlining the initial store setup process.
- **Inventory Settings Tracking:** The system now logs when merchants enable or disable inventory tracking and backorders within the inventory management views.

## **Bug Fixes**

- **Upsell Attribution Fix:** Resolved an issue where upsell transactions were missing initial attribution data and did not appear correctly when filtering payments by transaction source.
