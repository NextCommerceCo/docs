## New Features

- New Dashboard with a full suite of metrics to keep track of your store’s commerce, including:
- Total Sales
- Total Orders
- Avg Order Value
- New Customers
- Top Selling Products
- Top Converting Offers
- [Admin API](https://developers.nextcommerce.com/docs/admin-api) Create Order now supports Paypal as a payment method with a redirect flow.
- Admi API Create Order with User ID

## Improvements

- Shipping Address country now uses a Geo IP lookup to pre-select the user’s country if available.
