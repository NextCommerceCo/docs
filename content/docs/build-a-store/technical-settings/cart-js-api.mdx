---
title: "Storefront API & Events"
description: "Leverage the integrated GraphQL API and set up custom event trackers"
---
Set up custom event tracking from the **Settings > Tracking Events** menu.

The following Tracking Events are supported natively by Next Commerce storefronts:

* page\_viewed
* product\_category\_viewed
* product\_viewed
* product\_added\_to\_cart
* product\_removed\_from\_cart
* checkout\_started
* checkout\_contact\_info\_submitted
* checkout\_shipping\_address\_submitted
* checkout\_shipping\_method\_submitted
* checkout\_completed

For more information on Tracking Events and how to set up custom and external scripts, see our [**Developers portal**](https://developers.nextcommerce.com/docs/storefront/event-tracking).

<Callout type="warn">

Note: **Cart JS API** is now deprecated in favour of the storefront GraphQL API.\
\
See Developer Docs for more on GraphQL API and other storefront APIs.

</Callout>
