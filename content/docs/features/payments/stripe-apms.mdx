---
title: "Stripe APMs"
description: "Offer Alternative Payment Methods via your Stripe account"
---
Next Commerce supports several regional alternative payment methods both in the storefront checkout and on custom checkouts on the Admin API:

* Stripe Link
* Klarna Payments
* iDEAL
* Bancontact
* Sofort Banking
* Giropay
* SEPA Debit

These methods, along with [**Apple Pay**](/docs/features/payments/apple-pay) and [**Google Pay**](/docs/features/payments/google-pay) are available to enable as sub-methods under **Stripe** gateways, from the **Alternative Payment Methods** card.

Note - these methods may be available in specific regions only.  Consult Stripe's documentation to determine which methods are available in each market.

<Callout type="info">

For implementation details on custom checkouts via the Admin API, start with the [**External Checkout Flow**](https://developers.nextcommerce.com/docs/admin-api/guides/external-checkout), then use the payment-method guides for [**Link**](https://developers.nextcommerce.com/docs/admin-api/guides/link), [**Klarna**](https://developers.nextcommerce.com/docs/admin-api/guides/klarna), [**iDEAL**](https://developers.nextcommerce.com/docs/admin-api/guides/ideal), [**Bancontact**](https://developers.nextcommerce.com/docs/admin-api/guides/bancontact), and [**SEPA Direct Debit**](https://developers.nextcommerce.com/docs/admin-api/guides/sepa-debit).

</Callout>
